# Fundamentos de POO en JS; Mixins y otras tecnicas de reutilizacion

## La Plata JS / Meetup 6

Este repositorio contiene los ejemplos de la presentacion. 

##  Que necesitamos?
Para poder correr los ejemplos necesitamos tener instalado [Node](http://nodejs.org/)
Luego clonamos el repositorio e instalamos las dependencias con npm:

```
npm install
```

## Como ejecuto los ejemplos?

Los ejemplos estan numerados. Para correrlos:  

```
node {ejemplo}
```

### Ejemplos de Traits

La carpeta `cocktail-traits` contiene los ejemplos y demo de Traits con [CocktailJS](http://cocktailjs.github.io)
